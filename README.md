# Chat-App

A real-time chat application with rooms. This is taken from a tutorial that was
made by [TomDoesTech](https://www.youtube.com/watch?v=a_xo-SbIfUQ&list=PL0iFifR5umcmuQHltrvZQ3PU3i0IWZckE&index=3)

## Technologies used

**Node.js**

- Socket.io
- Express.js
- Typescript

**React.js**

- NextJS
- Socket.io-client
- Typescript
- CSS Modules
